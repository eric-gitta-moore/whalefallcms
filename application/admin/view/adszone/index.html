注意事项：

广告位ID 1~5为测试专用，正常使用时，需要新建广告位，也可以删掉ID为1~5的广告位，但是请勿直接使用，因为插件升级会删掉并重建ID为1~5的广告位

广告位调用说明：

可以参考API文档，使用接口调用，可以先用“php think api --force=1”命令生成接口文档后，点击查看接口文档；
电脑端控制器获取广告信息的参考代码： 
$Adszone = new \addons\adszone\library\Adszone();
$result = $Adszone->getAdsById($id); //按照ID调用广告位
$result = $Adszone->getAdsByMark($mark); //按照标记调用广告位
